1997 60-й Чемпионат мира по хоккею с шайбой 1996 года проходил в Австрии с 21 апреля по 5 мая . Сначала все сборные были поделены на две группы по шесть команд и играли по круговой системе. Затем четыре лучшие сборные из каждой группы играли по олимпийской системе на выбывание, начиная соответственно с четвертьфинала. Для определения третьего места проводился отдельный матч. Также проводились матчи для выявления команды, занявшей последнее место и выбывающей на будущий сезон из элиты. Чехия стала чемпионом мира, обыграв в финале сборную Канады со счетом 4-2 Содержание
